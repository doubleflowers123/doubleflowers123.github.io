### ***int***

*数据范围为 "-2147483648 ~ 2147483647"*

```hlsl
int C = A + B;

return C;
```

### ***half***

*这个也是整形 只不过 **int** 占4byte，**half**占2byte*，*为什么这么设计 因为在shader里面有时候不需要这么大的*

*half 范围在 "-60000~60000"*

```hlsl
return half(1);
```

### ***uint***

*取值范围是 "0 ~ 4294967295"*

